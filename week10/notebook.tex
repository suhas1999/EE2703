In this experiment we will be trying to do convolution operation on
various signals and using both methods of linear convolution and
circular convolution.In the theory class we have ana- lyzed the
advantages of using circular convolution over using linear convolution
when we are recieving continuos samples of inputs.We will also be
analysing the effect of passing the signal x = cos(0.2 ∗ pi ∗ n) +
cos(0.85 ∗ pi ∗ n) through a given filter.At last we will be analysing
the cross- correllation output of the Zadoff--Chu sequence.
